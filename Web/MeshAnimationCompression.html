INTRODUCTION

In this project, the following paper related to mesh animation compression was implemented:

Motion-aware Compression and Transmission of Mesh Animation Sequences

Bailin Yang et al, ACM Trans. Intell. Syst. Technol. 10, 3, Article 25 (May 2019), 21 pages

  • The paper proposed a method for vertex compression of mesh animation sequences with fixed connectivity between vertices (faces information) for all frames, such that the faces information of all frames are the same.

  • The main idea of this paper is to find the motion similarity of vertices of frames by grouping the frames with similar vertex motion strength (Temporal clustering) and finding the motion strength similarity of vertices inside of each frame (spatial segmentation).

  • The proposed mesh compression method is somewhat similar to the fundamental approach used in video compression where temporal and spatial redundancies are removed using inter-prediction and intra-prediction, respectively.

MESH ANIMATION COMPRESSION ALGORITHM

The proposed method for mesh animation compression has the following steps:

  1. Motion strength Computation

    2. In the first step , the motion strength of the vertices of all frames is computed based on the Curvature and Torsion of curves related to the trajectory of vertices over frames. The curvature and torsion of a curve show how sharply it is twisting in space. The positions of each vertex V in two consecutive frames t and t+1, make the given vector r.   2. Temporal Clustering

    3. A matrix including curvature and torsion of vertices is formed, where i and j show the indices of vertices and frames, respectively. Then, the rows of this matrix are clustered using the k-mean algorithm. In other words, frames are clustered in S groups based on similarities of curvature and torsion of their vertices.

  3. Spatial Segmentation

    4. The matrix of curvatures and torsions of vertices of all frames for each cluster is formed. The first three columns denote the average coordinates of the vertex i-th within a given cluster of frames. Each row shows the torsion and curvature values of all vertices (N) for all frames of a given cluster. The α value is a weight that determines the importance of each component. The rows of this matrix are grouped using the k-mean algorithm. In this case, vertices with similar motion strength inside each frame of the cluster are grouped (segmented).

  4. Compute Graph Fourier transform (GFT)

    5. GFT is a mathematical transform that decomposes the Laplacian Matrix of a graph into eigenvalues and eigenvectors. To compute the GFT coefficients of vertices, we first compute the Laplacian matrix (L) for the vertices of each segment. MESH ANIMATIOM COMPRESSION SOFTWARE

The mesh animation compression algorithm was implemented using C++ (Visual Studio 2019). The libraries used in the project are as follows:

  • Eigen for matrix computations.

  • GLEW library.

  • Imgui and nfd for creating controls (Button, TextBox, â€¦) in the OpenGL program.

Each step of the proposed mesh compression method is implemented in separate modules (classes).

The software has the following features:

  • Selecting the number of temporal clusters and spatial segments.

  • Loading the .obj file of a mesh animation with an optional number of frames and vertices.

  • Reconstructing the mesh animation with an optional compression factor

  • Computing and showing the KG-error after reconstruction.

  • Save the reconstructed frames in an optional folder.
